## What I Did

研修の一環として、Azureサービスを活用したクラウドインフラストラクチャを設計。Azure Application Gatewayのアーキテクチャとリクエストフローを学習し、Azure App ServiceとApp Service Planを適切な命名規則（Japan East/Westのリージョンサフィックス）で構成。Azure料金計算ツールを使用してコスト最適化戦略を提案した。

## Architecture

- **Compute**: Azure App Service + App Service Plan
- **Networking**: Azure Application Gateway（リクエストフローとルーティング）
- **Regions**: Japan East / Japan Westのマルチリージョン構成
- **Cost**: 料金計算ツールによる分析とコスト削減提案

## Technologies

Azure App Service, Azure Application Gateway, Azure Pricing Calculator
